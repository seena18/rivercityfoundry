# River City Web

A premium web design agency website built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

First, run the development server:

\`\`\`bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Home Page**: Landing page with services and CTA.
- **Contact**: Inquiry form.
- **Consultation**: Booking placeholder.

## Project Structure

- \`src/app\`: App router pages.
- \`src/components\`: Reusable components (Navbar, Footer).
